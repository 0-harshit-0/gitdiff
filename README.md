# gitdiff
A lightweight `git diff` wrapper in GO. It presents the difference in GUI format for better readability.

## build it
```bash
go build
```

## move it
```bash
mv gitdiff ~/.local/bin/ && chmod +x ~/.local/bin/gitdiff
```

## run it
```bash
gitdiff
```
